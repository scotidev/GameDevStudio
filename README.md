# GameDevStudio

This is a personal repository for AI instructions focused on game development. It is **not** intended for commercial use.

The repository will be updated and expanded over the long term as new instructions are written.

## Purpose

This repo contains personal instructions for creating games using various game engines. The instructions are organized by engine to make them easy to find and follow.

## Structure

| Folder | Description |
|--------|-------------|
| `Godot/` | AI instructions for game development with the Godot engine |
| `Unity/` | AI instructions for game development with the Unity engine |
| `Unreal/` | AI instructions for game development with the Unreal Engine |
